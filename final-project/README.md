# מנהל משימות (Todo Manager)

## הוראות הרצה

1. התקנת ספריות:

```bash
npm install
```

2. הרצת הפרויקט במצב פיתוח:

```bash
npm run dev
```

3. פתיחת הדפדפן בכתובת שמוצגת בטרמינל ל `http://localhost:5173`)

## מבנה הקומפוננטות

### App.jsx

הקומפוננטה הראשית המנהלת את state של כל המשימות, הפילטר הנוכחי, וכל הפונקציות לניהול המשימות (הוספה, מחיקה, עריכה, שינוי סטטוס). אחראית גם על שמירה וטעינה מ-LocalStorage.

### TaskInput.jsx

קומפוננטה המטפלת בקלט מהמשתמש להוספת משימות חדשות. כוללת שדה טקסט וכפתור הוספה, ותומכת בהוספה באמצעות Enter.

### ListItem.jsx

קומפוננטה המציגה את רשימת המשימות כולה ומרנדרת קומפוננטה TodoItem לכל משימה.

### TodoItem.jsx

קומפוננטה המייצגת משימה בודדת ברשימה. כוללת checkbox לסימון הושלמה, שדה טקסט לעריכה, וכפתור מחיקה.

### FilterControls.jsx

קומפוננטה המטפלת בסינון המשימות. מציגה שלושה כפתורים: הכל, פעיל, והושלם, ומסמנת ויזואלית את הפילטר הנוכחי.
