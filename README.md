# מנהל משימות (Todo Manager)

אפליקציית ניהול משימות מודרנית ונוחה לשימוש, הבנויה ב-React עם עיצוב כהה ואלגנטי.

## תכונות עיקריות

- **הוספת משימות** - הוסף משימות חדשות בקלות
- **עריכת משימות** - ערוך משימות קיימות בלחיצה כפולה או כפתור עריכה
- **מחיקת משימות** - מחק משימות שאינן נחוצות
- **סימון הושלמה** - סמן משימות כהושלמו
- **סינון משימות** - צפה בכל המשימות, רק פעילות, או רק הושלמו
- **ניקוי משימות הושלמו** - מחק את כל המשימות שהושלמו בבת אחת
- **שמירה אוטומטית** - כל המשימות נשמרות אוטומטית ב-LocalStorage
- **הודעות שגיאה** - הודעות ברורות למשתמש במקרה של שגיאות

## עיצוב

האפליקציה מציגה עיצוב מודרני ומינימליסטי עם:

- **רקע כהה** - רקע שחור ואפור כהה לעיניים נוחות
- **צבעים עיקריים** - אפור ושחור כצבעים דומיננטיים
- **אדום לדברים קיצוניים** - כפתורי מחיקה ופעולות הרסניות מודגשים באדום
- **אנימציות חלקות** - מעברים חלקים ואלגנטיים בין מצבים
- **עיצוב רספונסיבי** - מותאם לכל הגדלי מסך

## התקנה והרצה

### דרישות מוקדמות

- Node.js (גרסה 16 ומעלה)
- npm או yarn

### שלבי התקנה

1. **התקן את התלויות:**
   ```bash
   npm install
   ```

2. **הרץ את השרת לפיתוח:**
   ```bash
   npm run dev
   ```

3. **פתח את הדפדפן:**
   - האפליקציה תרוץ על `http://localhost:5173` (או פורט אחר שמוצג בטרמינל)
   - הדפדפן יפתח אוטומטית

### בנייה לייצור

לבניית גרסת ייצור:

```bash
npm run build
```

הקבצים ייבנו לתיקיית `dist` שניתן לפרוס על כל שרת סטטי.

## הסבר על הקומפוננטות

### App.jsx
הקומפוננטה הראשית המנהלת את כל הלוגיקה של האפליקציה:
- ניהול state של המשימות
- פונקציות להוספה, מחיקה, עריכה ושינוי סטטוס
- שמירה וטעינה מ-LocalStorage
- ניהול פילטרים והודעות שגיאה

### TaskInput.jsx
קומפוננטה לטיפול בקלט מהמשתמש:
- שדה טקסט להוספת משימה חדשה
- כפתור הוספה
- תמיכה בהוספה באמצעות Enter

### ListItem.jsx
קומפוננטה המרנדרת את רשימת כל המשימות:
- מעבר על רשימת המשימות
- העברת props לכל TodoItem

### TodoItem.jsx
קומפוננטה המייצגת משימה בודדת:
- Checkbox לסימון הושלמה
- מצב עריכה עם שדה טקסט
- כפתורי עריכה ומחיקה
- עיצוב דינמי לפי סטטוס המשימה

### FilterControls.jsx
קומפוננטה לניהול סינון המשימות:
- כפתורים לסינון: הכל, פעיל, הושלם
- הדגשה ויזואלית של הפילטר הפעיל

### ErrorMessage.jsx
קומפוננטה להצגת הודעות שגיאה:
- הודעה בולטת באדום
- כפתור סגירה
- נעלמת אוטומטית אחרי 3 שניות

## שימוש

### הוספת משימה
1. הקלד את הטקסט של המשימה בשדה הקלט
2. לחץ על כפתור "הוסף" או לחץ Enter

### עריכת משימה
1. לחץ על הטקסט של המשימה (אם היא לא הושלמה)
2. או לחץ על כפתור "ערוך"
3. ערוך את הטקסט
4. לחץ Enter או לחץ מחוץ לשדה לשמירה

### מחיקת משימה
לחץ על כפתור "מחק" ליד המשימה

### סימון משימה כהושלמה
לחץ על ה-checkbox ליד המשימה

### סינון משימות
לחץ על אחד מכפתורי הסינון:
- **הכל** - מציג את כל המשימות
- **פעיל** - מציג רק משימות שלא הושלמו
- **הושלם** - מציג רק משימות שהושלמו

### ניקוי משימות הושלמו
לחץ על כפתור "נקה משימות הושלמו" למחיקת כל המשימות שהושלמו

## טכנולוגיות

- **React** - ספריית UI
- **Vite** - כלי בנייה מהיר
- **LocalStorage** - שמירה מקומית של נתונים
- **CSS3** - עיצוב מודרני עם gradients ו-animations

## הערות

- כל המשימות נשמרות אוטומטית ב-LocalStorage של הדפדפן
- המשימות נטענות אוטומטית בעת טעינת הדף
- הודעות שגיאה נעלמות אוטומטית אחרי 3 שניות
- לא ניתן לשמור משימה ריקה
